Import / Export

Outils

Les outils mongodump et mongorestore permettent respectivement d'effectuer des backups de base de données et de les restaurer.

Il existe également les outils mongoexport et mongoimport qui permettent de faire des imports / exports aux formats JSON ou CSV.

Outils d'import/export MongoDB

Import

  1. Importez la base de données citybike :

    mongorestore --drop --gzip --archive=./citybike.gz

    Cette base de données contient l'ensemble des données des stations vélo de la ville de Rennes ainsi qu'une liste de 1000 utilisateurs.

Export

La commande ci-dessous permet d'effectuer un export :

mongodump --gzip --drop --db citybike --archive=./
